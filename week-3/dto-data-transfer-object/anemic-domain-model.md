# 🟨 빈약한 도메인 모델(anemic domain model)

도메인 객체들에 비즈니스 로직이 거의없거나 아예 없는 소프트웨어 도메인 모델을 의미한다.\
예시.

DTO의 경우, 비지니스 로직이 들어가거나 객체간의 협력 개념 보단, 전송의 목적이기 때문에 해당 안티패턴에 들어가긴 한다.
